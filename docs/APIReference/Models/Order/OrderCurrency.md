# OrderCurrency

## Descripción

Muestra l amoneda del pedido. Forma parte de Order en forma de lista.

## Métodos

- **Integer** getId()
- ***CurrencyMode*** getMode()
- **int** getCurrencyId()
- **String** getName()
- **String** getCode()
- **String** getCodeNumber()
- **String** getSymbol()
- **double** getUsdValue()

## Referencias

- **CurrencyMode**: Enumerado
