# UserAddress

## Descripción

Define la dirección del usuario. Ya sea como *billing* o como *shipping*. Extiende de **[Address](Address.md)**.

## Métodos

- **boolean** isDefaultAddress()
- **String** getAlias()
- **String** getFirstName()
- **String** getLastName()
- **String** getCompany()
- **String** getAddress()
- **String** getAddressAdditionalInformation()
- **String** getNumber()
- **String** getVat()
- **String** getNif()
- **String** getFax()
- **boolean** isTax()
- **boolean** isRe()
- **boolean** isReverseChargeVat()
