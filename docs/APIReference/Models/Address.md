# Address

## Descripción

Muestra la dirección de un usuario o del usuario del pedido. La dirección puede ser de facturación o de envío según venga definido por el usuario.

## Métodos

- **String** getName()
- **String** getAddress()
- **String** getCity()
- **String** getState()
- **String** getPostalCode()
- **[Location](Location.md)** getLocation()
- **String** getPhone()
- **String** getMobile()
