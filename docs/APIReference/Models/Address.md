# Address

## Descripción

Muestra la dirección de un usuario o del usuario del pedido. La dirección puede ser de facturación o de envío según venga definido por el usuario.

## Métodos

- **String** getAlias()
- **String** getFirstName()
- **String** getLastName()
- **String** getCompany()
- **String** getAddress()
- **String** getAddressAdditionalInformation()
- **String** getNumber()
- **String** getCity()
- **String** getState()
- **String** getPostalCode()
- **String** getVat()
- **String** getNif()
- **[Location](Location.md)** getLocation()
- **String** getPhone()
- **String** getMobile()
- **String** getFax()
- **boolean** isTax()
- **boolean** isRe()
- **boolean** isReverseChargeVat()
