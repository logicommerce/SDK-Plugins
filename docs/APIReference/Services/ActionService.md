# ActionService

Servicio que no tiene métodos, por lo tanto no tiene un uso directo.
Necesario para agrupar los servicios de tipo "acción".

- **[OrderActionService](./OrderActionService.md)** : Acciones relacionadas con Pedidos.
- **[ShipmentActionService](./ShipmentActionService.md)** : Acciones relacionadas con envíos.


## Interfaz

```java

public interface ActionService extends PluginService {

}


```
